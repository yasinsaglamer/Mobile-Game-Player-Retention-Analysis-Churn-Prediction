# Mobile Game Player Retention Analysis & Churn Prediction

This project focuses on analyzing player behavior in a mobile game environment, specifically investigating the impact of "gate" placement on retention and predicting player churn using machine learning.

## 🚀 Project Overview
In this study, I performed a comprehensive data analysis on a mobile game dataset to optimize player engagement. The project is divided into two main analytical phases:
1. **A/B Testing:** Statistical analysis of how moving a gate from level 30 to level 40 affects player retention.
2. **Predictive Modeling:** Utilizing Support Vector Machines (SVM) to predict whether a player will churn or remain active based on their early-game behavior.

## 🛠️ Key Features
* **Exploratory Data Analysis (EDA):** Visualizing player distributions, spending habits, and identifying bottlenecks in level progression.
* **Statistical Inference:** Implementing **Mann-Whitney U** tests and **Bootstrap** methods to validate A/B test results with high confidence.
* **Machine Learning:** Building an **SVM (Support Vector Machine)** classifier with scaled features and balanced class weights to handle churn prediction.
* **Visualizations:** Detailed plots using `Seaborn` and `Matplotlib`, including KDE plots for bootstrap distributions and Confusion Matrices for model evaluation.

## 📊 Technical Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, SciPy
* **Environment:** Jupyter Notebook (.ipynb)
